# Customer API

A Spring Boot REST API for managing customer records, built with a layered architecture (Controller → Service → Repository), backed by Oracle Database, and documented with Swagger/OpenAPI.

## Tech Stack

- Java 17
- Spring Boot (Web, Data JPA, Validation)
- Oracle Database (Free/XE, via Docker)
- springdoc-openapi (Swagger UI)
- Lombok
- Maven

## Architecture

Controller → handles HTTP requests, uses DTOs only
Service → business logic, maps between DTOs and Entities
Repository → Spring Data JPA interface, talks to the database
Entity → JPA-mapped class, internal to the persistence layer
DTO → request/response shape exposed via the API

## Prerequisites

- Java 17+
- Maven (or use the included `./mvnw` wrapper)
- Docker (for Oracle Database)

## Database Setup

1. Start the Oracle container:

   docker start oracle-free

2. Create a dedicated app user:

   docker exec -it oracle-free sqlplus / as sysdba

   ALTER SESSION SET CONTAINER = FREEPDB1;
   CREATE USER customer_app IDENTIFIED BY "CustomerAppPass123";
   GRANT CONNECT, RESOURCE TO customer_app;
   GRANT UNLIMITED TABLESPACE TO customer_app;

3. Create the customer table:

   docker exec -it oracle-free sqlplus customer_app/CustomerAppPass123@localhost:1521/FREEPDB1

   CREATE TABLE customer (
       id            NUMBER(19)      GENERATED BY DEFAULT AS IDENTITY PRIMARY KEY,
       full_name     VARCHAR2(150)   NOT NULL,
       email         VARCHAR2(150)   NOT NULL UNIQUE,
       phone_number  VARCHAR2(30),
       address       VARCHAR2(255),
       created_at    TIMESTAMP DEFAULT SYSTIMESTAMP NOT NULL
   );

## Configuration

Connection details live in `src/main/resources/application.properties`:

    spring.datasource.url=jdbc:oracle:thin:@localhost:1521/FREEPDB1
    spring.datasource.username=customer_app
    spring.datasource.password=CustomerAppPass123
    spring.datasource.driver-class-name=oracle.jdbc.OracleDriver
    spring.jpa.database-platform=org.hibernate.dialect.OracleDialect
    spring.jpa.hibernate.ddl-auto=validate
    server.port=8081

## Running the App

    ./mvnw spring-boot:run

The app starts on http://localhost:8081.

## API Documentation (Swagger)

Once running, open:

    http://localhost:8081/swagger-ui.html

## Endpoints

| Method | Path                  | Description                  |
|--------|-----------------------|-------------------------------|
| POST   | /api/customers         | Create a new customer         |
| GET    | /api/customers/{id}    | Retrieve a customer by ID     |
| GET    | /api/customers         | Retrieve all customers        |
| PUT    | /api/customers/{id}    | Update an existing customer   |
| DELETE | /api/customers/{id}    | Delete a customer             |
